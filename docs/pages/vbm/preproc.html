Voxel-based Morphometry - Preprocessing

Voxel-based morphometry (VBM) is a neuroimaging technique that investigates focal differences in brain anatomy. The core process of VBM is segmenting the brain into grey matter, white matter, and cerebrospinal fluid, warping the segmented images to a template space and smoothing. Thereafter, statistical analysis is performed on the basis of the general linear model. Some resources to explore for better understanding:

  • Resource 1: Subscribe to SPM MAIL which is an electronic mailing list for discussion and help with the methodology, implementation and use of Statistical Parametric Mapping and the SPM package. I have got to learn a lot from these mails, and sometimes read mails from early 2000s to find my answers!
  • Resource 2: Watch the YouTube video on VBM from the makers of SPM12. A crash-course on VBM Fundamentals.
In this tutorial, we will be covering the procedure of conductin VBM using CAT12 and SPM12. For this tutorial, you need to download CAT12 toolbox. Now, I will provide a brief overview of all the steps involved:

  • Using CAT12 graphical interface (GUI), we will implement the default SEGMENT pipeline to segment the raw T1 images into GM and WM maps.
  • We will exclude subjects with poor Image Quality Rating (IQR) using CAT12 QC Reports. This step is very important to remove outliers which can influence your results.
  • We will use the SPM12 SMOOTH function to smooth our GM maps (i.e., mwp1* - files starting with this prefix are segmented GM images).
  • We will build full-factorial model using BASIC MODELS function of CAT12. This step is very important as we define our model, contrasts, covariates (e.g., Age, Gender, TIV).
  • We will use ESTIMATE function of SPM12 to estimate the SPM.mat, which is your design file.
  • Now, we are all set to view our results. We only need to define cluster-forming and peak-level thresholds to correct for multiple comparisons.
  • We will use xjview and Surf Ice to visualise our results.
As you can see, we have used multiple softwares in our VBM workflow, it is very important to keep your files and directory structured.
